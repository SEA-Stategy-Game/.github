## Organization Structure

Each development group maintains its own dedicated repository for focused, parallel development:

- **AI / Planning** – Responsible for AI systems, pathfinding, game state planning and strategic decision-making.
- **Core Systems** – Responsible for the foundational game engine systems, data management and core gameplay mechanics.
- **Multiplayer** – Responsible for networking infrastructure, synchronization and multiplayer game modes.
- **Interface** – Responsible for the user interface, menus, HUD and overall player-facing experience.

The **Client Project** repository brings together the work of all groups into the final, integrated game client.
